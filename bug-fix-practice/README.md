# Bug Fix Practice

This folder contains several frontend bug-fix practice projects focused on layout, spacing, and responsive UI issues.

## Projects

- `button-alignment-fix/`
- `contact-form-ui-fix/`
- `faq-accordion-fix/`
- `faq-fix/`
- `mobile-nav-fix/`

## How to use

Open a folder and launch its `index.html` file in a browser to inspect the layout and behavior.

## Purpose

Each subfolder is designed as a small repair exercise for common frontend issues like mobile overflow, form spacing, and accordion state.
