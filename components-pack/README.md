# Components Pack

This folder contains reusable UI components built with HTML, CSS, and JavaScript.

## Components

- `contact-form-ui/`
- `faq-accordion/`
- `hero-section/`
- `navbar/`

## How to use

Open a component folder and launch `index.html` in a browser to preview the isolated UI pattern.

## Purpose

The components in this pack are meant for frontend development practice, design systems, and responsive UI building.
