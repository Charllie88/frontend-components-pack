# Business Website

A modern, responsive business website featuring home page, services page, and contact page with professional design and functionality.

## Pages

### Home Page (`index.html`)

- **Hero Section**: Eye-catching introduction with statistics and call-to-action buttons
- **Services Preview**: Showcase of key business services with hover effects
- **Testimonials**: Customer reviews with star ratings
- **CTA Section**: Call-to-action for contact
- **Footer**: Comprehensive footer with links and social media

### Contact Page (`contact.html`)

- **Contact Form**: Interactive form for inquiries with validation
- **Address & Hours**: Business location and operating hours
- **Social Links**: Social media follow buttons
- **Map Placeholder**: Interactive map location display

### Services Page (`services.html`)

- **Detailed Service Cards**: Comprehensive descriptions of consulting, implementation, and analytics services
- **Pricing Blocks**: Three pricing tiers (Starter, Professional, Enterprise) with feature comparisons
- **FAQ Section**: Expandable FAQ accordion with common questions
- **CTA Section**: Call-to-action for consultations and contact

## Technologies Used

- HTML5
- CSS3 (with modern features like CSS Grid, Flexbox, and animations)
- JavaScript (for interactivity, form validation, and animations)
- Google Fonts (Inter font family)

## Features

- Fully responsive design for all devices
- Smooth scrolling navigation
- Mobile-friendly hamburger menu
- Form validation and submission handling
- Interactive FAQ accordion
- Pricing comparison cards
- Professional business layout

## Getting Started

1. Clone or download the project files
2. Open `index.html` in a web browser to view the home page
3. Navigate to `services.html` for detailed services and pricing
4. Navigate to `contact.html` for the contact page
5. All functionality works without a server
