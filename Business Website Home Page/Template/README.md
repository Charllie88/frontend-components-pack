# Template Pack

Standalone business template package with the following sections:

- Hero section
- Pricing section
- FAQ accordion
- Testimonials section

## Files

- `index.html` - Template markup
- `styles.css` - Self-contained styling and responsive layout
- `script.js` - Mobile nav and FAQ accordion behavior

## Notes

- The package is standalone and does not depend on the main site files.
- Update the brand name, pricing copy, and testimonials directly in `index.html`.
- Colors and spacing are controlled from the CSS variables at the top of `styles.css`.
